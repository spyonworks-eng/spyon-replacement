Google Sheets — Two-Way Cloud Sync

Setup Instructions (one time)

1. Go to sheets.google.com → create a new sheet → name the first tab: REPLACEMENT LOG
2. Click Extensions → Apps Script → delete existing code → paste the code below → Save
3. Click Deploy → New deployment → type: Web app → Execute as: Me → Access: Anyone → Deploy
4. Authorise → copy the Web App URL → paste below → click Connect

⬆ Push All Changes — sends only new or changed slips to Sheets (smart diff, never wipes). Also pushes contacts and items.
⬇ Pull All — pulls slips, contacts and items from Sheets and merges into this device.
